# Business Model Canvas - Tech4Value

**Version 2.0 (Consolidation)**
**Système de Pilotage Stratégique pour Gestion de Portefeuille Projets**
*Projet d'Étude M2 CPIT 2025*

---

## Vue d'Ensemble

Ce Business Model Canvas positionne **Tech4Value** comme une plateforme de **pilotage stratégique et de gouvernance** capable d'intégrer les systèmes d'information existants (Odoo, SharePoint, Power BI, Azure AD) et de centraliser la gestion de portefeuille de 5 projets critiques pour 185 collaborateurs.

La plateforme automatise les reportings (+30% de temps libéré), améliore la prise de décision via données consolidées temps-réel, et substitue 7 outils fragmentés par 1 solution unifiée.

**Contexte client** : Entreprise de 185 collaborateurs gérant 5 projets stratégiques majeurs mobilisant 78 ETP (42% des effectifs).

❌ **Resistance au changement** - Habitudes consolidées par pôle
❌ **Coûts d'intégration** - Synchronisation Odoo/Excel complexe (risque technique)

### Opportunités (Opportunities)

🚀 **Innovation produit** - Dashboards prédictifs, automations IA, marketplace templates

### Menaces (Threats)

⚠️ **Turnover équipe** - Perte talents critiques mid-implementation (dépendance personnes)

## ANNEXE B : Matrice Octalysis - Core Drives

**Application du framework de Yu-Kai Chou à Tech4Value**

| Core Drive | Segment | Mécanique Appliquée |
|-----------|---------|---------------------|
| **CD1: Epic Meaning** | Tous | Narrative "vous pilotez Tech4Value" + Legacy documenté + Badges permanents "Architect of [Project]" |
| **CD2: Accomplishment** | Chefs de Projet, Teams | Levels (Rookie→Lead PM), XP progression, Milestone celebrations, "Top 10% Performers" |
| **CD3: Empowerment** | PMO, Chefs de Projet | Customization UI, workflow automation no-code, AI suggestions, Template Marketplace |
| **CD4: Ownership** | Tous | Portfolios personnels, template collections, badge gallery, CV dynamique auto-généré |
| **CD5: Social Influence** | Tous | Leaderboards équipe (non individuel), peer recognition, mentorat, community forums |
| **CD6: Scarcity** | Tous (modéré) | Time-limited challenges, appointment slots, early-access features (éthique) |
| **CD7: Curiosity** | Tous (modéré) | Mystery insights IA, surprise feedback, serendipity discoveries, "Easter eggs" |
| **CD8: Loss Avoidance** | Executives, Managers | Risk dashboards, deadline countdowns, compliance alerts, escalations automatiques |

**Équilibrage Éthique** :
- **70% White Hat (CD1-5)** : Motivation intrinsèque, sens, accomplissement, créativité
- **30% Black Hat (CD6-8)** : Avec garde-fous éthiques (opt-out, stress monitoring, A/B testing)

**Mécaniques Anti-Stress** :
- Opt-out gamification disponible pour tous utilisateurs
- Monitoring sentiment utilisateur via surveys régulières
- A/B testing systématique avant déploiement mécaniques engagement
- Leaderboards équipe uniquement (jamais individuel pour éviter compétition toxique)
