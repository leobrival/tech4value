## Légende & Méthodologie

### Lecture du Tableau

**OBJECTIF** : Résultat stratégique visé (aligné OKRs et vision produit)

**KPI** : Indicateurs mesurables avec **baseline → cibles intermédiaires → cible finale**

- Format : `Métrique : Baseline → Cible S6 → Cible S12 → Cible S24`
- Semaines indiquées entre parenthèses (S1-S24)

**ACTIONS** : Initiatives concrètes avec timeline

- Format : `Action (détails, Semaines)`
- Priorisées selon scoring RICE (Reach × Impact × Confidence / Effort)
