## Notes Importantes

### Distribution des Notes
- **Notes de groupe** : La même note est attribuée à tous les membres du groupe
- **Notes individuelles** : Chaque étudiant est évalué sur sa contribution personnelle

### Conseils pour Maximiser les Points

#### Pour la Vidéo
- Soignez la structure et la fluidité
- Assurez-vous que chaque membre parle (avec affichage du nom)
- Démontrez clairement la valeur ajoutée du MVP
- Adoptez une posture professionnelle

#### Pour le Dossier Technique
- Utilisez des outils professionnels (Gantt, RACI, etc.)
- Justifiez tous vos choix techniques
- Intégrez les aspects GreenIT, accessibilité et conformité
- Proposez une vision d'évolution à long terme
- Soignez la documentation utilisateur

#### Pour l'Analyse Individuelle
- Soyez honnête sur les défis rencontrés
- Démontrez votre capacité d'auto-critique
- Identifiez clairement vos apprentissages
- Proposez des axes d'amélioration concrets
